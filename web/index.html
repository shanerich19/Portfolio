ABOUT CODING

My websites always begin as a concept on a piece of paper. Before I even begin designing the website, I have to understand my target audience. The site must be continuitous with the brand or else there is a visual disconnect that can be confusing to the user. Once everything is sorted, the main layout is chosen as well as the choice of fonts and colors. Each page is designed with the user in mind, this means no confusing links, images, or text. It has to be as easy for a first time user as it is a common visitor. Again, simplicity is important.
